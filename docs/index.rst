***************
Django Workshop
***************

*Django Workshop* is a free tutorial for the Python open source web
framework `Django <http://www.djangoproject.com/>`_ |djangoversion|
which has been written by :doc:`Markus Zapke-Gr√ºndemann and others
<authors>`. It is suitable for people who have basic programming skills.
You can either use the tutorial for self-study or `book a training
<http://www.transcode.de/>`_ if you are stuck or need information not
provided in this tutorial.

Free means the tutorial is distributed under a :doc:`Creative Commons License
<copyright>` and everyone is invited to `contribute
<https://github.com/keimlink/django-workshop>`_. If you have any suggestions
for improvements, or find any errors, please let us know in the `Issue Tracker
<https://github.com/keimlink/django-workshop/issues>`_.

If you have basic knowledge in the following areas it will be a lot
easier for you to learn Django:

    - `Object-oriented programming (using Python) <http://learnpythonthehardway.org/>`_
    - `HTTP Request/Response model <https://en.wikipedia.org/wiki/Hypertext_Transfer_Protocol>`_
    - `HTML and CSS <http://htmldog.com/guides/>`_
    - `SQL and relational databases <http://sqlzoo.net/>`_
